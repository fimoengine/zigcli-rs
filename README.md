# zigcli

A build dependency for running the `zig` build tool to compile a native
library.

```toml
# Cargo.toml
[build-dependencies]
zigcli = "0.1.0"
```

The Zig executable is assumed to be `zig` unless the `ZIG`
environmental variable is set.
